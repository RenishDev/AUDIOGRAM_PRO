## 🚀 HOW TO USE

### Print Single Report
```
1. Navigate to patient record
2. Click "Print" button
3. Review in print preview
4. Click "Save as PDF" or "Print"
```

### Print Bulk Reports
```
1. Go to Active Clinical Database
2. Check boxes next to patients
3. Click "Bulk Print Selected"
4. Review in print preview
5. Click "Save as PDF" or "Print"
```

### Save as PDF
```
1. In print preview, press Ctrl+S (Windows) or Cmd+S (Mac)
2. Choose save location
3. Click "Save"
```
