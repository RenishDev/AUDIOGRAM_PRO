🚀 START HERE

Step 1: Open the Application
    → Visit: http://localhost:9002/

Step 2: View a Patient Record
    → Click on any patient in the database
    → Or go to: http://localhost:9002/test/ivn213o (test patient)

Step 3: Print a Single Report
    → Click the "Print" button
    → Preview appears
    → Press Ctrl+S to save as PDF, or click "Print"

Step 4: Bulk Print Multiple Reports
    → Go to "Clinical Database" section
    → Check boxes next to patient names
    → Click "Bulk Print Selected"
    → All reports appear in print preview
    → Each report on separate page
    → Save as PDF or print
