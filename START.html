🚀 Quick Start

Your offline AudiogramPro application is ready to run! Choose one of the methods below:

⚡ Easiest Method (Windows)

Simply double-click the run.bat file:

1

Double-Click run.bat

Look for run.bat in the project folder

2

Wait for Installation

It will install dependencies if needed (first run only)

3

Browser Opens Automatically

Your app will open at http://localhost:9002

💻 PowerShell Method (Windows)

Run this command in PowerShell:

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser cd "d:\setup\laragon\www\LAB" .\run.ps1

🖥️ Terminal Method (All Platforms)

Run these commands in Terminal/CMD:

cd "d:\setup\laragon\www\LAB" npm install --omit=optional npm run dev

Then open: http://localhost:9002

✅ Pro Tip: Use run.bat for the easiest experience. It handles everything automatically!
